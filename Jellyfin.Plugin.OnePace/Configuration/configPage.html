One Pace Plugin Settings

This plugin provides metadata integration for the One Pace project — a fan edit of One Piece that removes filler and improves pacing.


The language to use for arc and episode names/descriptions when available from the One Pace API.

The full path to your One Pace folder on the server. Any media under this path will be identified as One Pace content, even without [One Pace] tags in file/folder names.

Library Setup

Organize your One Pace files in the following folder structure:

/media/anime/
├── One Pace/
│   ├── [One Pace][1-7] Romance Dawn [1080p]/
│   │   ├── [One Pace][1] Romance Dawn 01 [1080p][D767799C].mkv
│   │   └── [One Pace][2] Romance Dawn 02 [1080p][04A43CEF].mkv
│   ├── [One Pace][237-303] Skypiea [720p]/
│   │   ├── [One Pace][237-238] Skypiea 01 [720p][En Dub][5B9B150E].mp4
│   │   └── [One Pace][239-240] Skypiea 02 [720p][En Dub][ABCD1234].mp4

Important: Enable the "One Pace" downloaders for Metadata (Shows, Seasons, Episodes) and Image fetchers in your library settings, and move them to the top.
